# Frontend Application for Traffic Data Visualization

This frontend application is built using React with Vite and styled with Tailwind CSS. It provides a user interface to visualize traffic data for four districts, allowing users to interact with the data and view detailed diagrams.

## Project Structure

- **src/**: Contains the source code for the React application.
  - **components/**: Contains React components.
    - **District.jsx**: Component that displays traffic data for a specific district.
  - **App.jsx**: Main application component that sets up the layout and routing.
  - **main.jsx**: Entry point for the React application.

- **index.html**: The main HTML file that serves the frontend application.

- **package.json**: Configuration file for the frontend, listing dependencies and scripts.

- **tailwind.config.js**: Configuration for Tailwind CSS.

- **vite.config.js**: Configuration for Vite.

## Getting Started

1. **Install Dependencies**: Run the following command in the `frontend` directory to install the required packages:
   ```
   npm install
   ```

2. **Run the Application**: Start the development server with:
   ```
   npm run dev
   ```

3. **Open in Browser**: Navigate to `http://localhost:3000` to view the application.

## Features

- Displays traffic data categorized as high, medium, or low for each district.
- Interactive district components that allow users to click and view detailed diagrams of traffic data.

## Technologies Used

- **React**: JavaScript library for building user interfaces.
- **Vite**: Build tool that provides a fast development environment.
- **Tailwind CSS**: Utility-first CSS framework for styling.
